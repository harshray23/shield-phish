# ShieldPhish Technology Stack

This document outlines the technology stack used in the ShieldPhish application.

| Category | Technology |
| :--- | :--- |
| **Languages**| TypeScript, CSS, HTML |
| **ML/AI Frameworks**| Genkit, Google Gemini |
| **Web Frameworks** | Next.js, React |
| **Frontend** | Tailwind CSS, ShadCN UI, Lucide React |
| **Parsing & Data** | Zod (for schema validation) |
| **Database** | Firebase Firestore |
| **Authentication** | Firebase Authentication |
| **Knowledge Base** | The AI model's knowledge is informed by public security data, including sources like PhishTank, Kaggle Phishing Datasets, and safe site lists (e.g., Alexa/Tranco). |
